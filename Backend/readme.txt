Getting Started
Steps:
1: Open Backend Folder in Terminal
2: if Virtual Environment folder .venv is not created, create it using " py -3 -m venv .venv "
3: Activate Virtual Environment using " .venv\Scripts\activate "
4: Make sure flask is installed using " pip install flask " and " pip install flask-cors "
5: Set Environment Variable " $env:FLASK_APP = "index" "
6: Run the application using " flask --app index run "
7: Run in Debug mode with " python index.py "
